# Prompt optimisé pour Hostinger Horizon

## Objectif
Utiliser ce prompt dans le chat de création sur **hostinger.fr** afin que l'IA génère automatiquement l'architecture et la base de données d'une application VTC complète.

## Prompt prêt à l'emploi
```
Je veux créer une application web complète de type VTC (comme Uber).
Fonctionnement (Logique Input/Output) : L'utilisateur doit pouvoir entrer un point de départ et une destination. L'application doit ensuite calculer un prix estimé et proposer 3 gammes de véhicules (Éco, Berline, Van). Une fois la commande validée, l'écran doit afficher une interface de suivi de course (tracking) avec le statut du chauffeur.
Design & Style (Instructions strictes) : Je veux un design ultra-professionnel 'Corporate Tech'.
• Important : N'utilise absolument aucun émoji ou smiley (cela fait trop scolaire). Utilise uniquement des icônes vectorielles modernes et fines.
• Couleurs : Fond blanc épuré, typographie noire très lisible, et boutons d'action en noir profond ou bleu nuit pour un fort contraste.
• Carte : L'arrière-plan principal doit être une carte interactive (Map) couvrant tout l'écran, avec les menus qui se superposent en bas (style 'Bottom Sheet').
```

## Pourquoi ce prompt fonctionne
- Il définit clairement les **inputs** (départ/destination) et les **outputs** attendus (prix, gammes de véhicules, suivi de course), ce qui aide l'IA à structurer les flux métier.
- Il impose un style **sans emoji** dès le départ pour éviter des itérations de correction de design.
- Il précise l'usage d'une **carte plein écran** avec une interface **bottom sheet**, ce qui cadre l'UX et la hiérarchie visuelle.
